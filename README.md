# Career Mate 🧑‍💼💡  
**생성형 AI 기반 커리어 · 채용 공고 추천 서비스**

## 1. 프로젝트 개요

Career Mate는 **사용자의 이력서 정보를 분석하여 적합한 채용 공고를 추천**하고,  
기업 정보·채용 공고·게시글을 통합적으로 제공하는 **커리어 매칭 서비스**입니다.

- 이력서 기반 **AI 직무·채용 공고 추천**
- 기업 및 채용 공고 조회
- 커뮤니티 게시글 기능
- 사용자 인증 및 프로필 관리

---

## 2. 기술 스택

## 5. 추천 알고리즘에 대한 기술적 설명

### 🔍 전체 흐름
1. 사용자가 이력서(PDF파일)를 등록
2. 서버에서 이력서 내용을 기반으로 AI 분석 요청
3. 생성형 AI가 다음 정보를 생성
   - 적합한 세 개의 채용 공고 선정
   - 추천 사유 (자연어 설명)
4. 추천 결과를 DB에 저장
5. 프론트엔드에서 추천 결과 조회

### 🧠 AI 추천 방식
- **생성형 AI 기반 학습된 텍스트 분석**
- gemini-2.0-flash 모델 사용
- 이력서의 기술 스택, 경험, 키워드를 중심으로 분석
- "Dacon-잡케어 추천 알고리즘 경진대회"에서 AI모델 학습에 사용된 사용자의 이력서 정보와 직무/기업 정보의 매칭에 대한 기록을 프롬프팅에 활용하여 간단한 학습 진행
- 채용 공고 설명과의 **의미적 유사성**을 기준으로 추천

## 6. 핵심 기능 설명

### 1️⃣ 이력서 분석 & 채용 공고 추천
- 사용자가 업로드한 이력서 파일을 AI가 분석
- 추천 공고와 함께 **설명 가능한 추천 결과** 제공

### 2️⃣ 기업 및 채용 공고 관리
- 기업 정보 상세 페이지
- 기업별 채용 공고 목록 제공

### 3️⃣ 기업 커뮤니티 게시글
- 기업별 게시글 및 댓글 CRUD
- 로그인 사용자 기반 권한 관리
- 사용자–기업 간 간접적 관계 형성

### 4️⃣ 사용자 인증
- JWT 기반 로그인/로그아웃

## 7. 생성형 AI 활용 부분

| 활용 영역 | 설명 |
|---|---|
| 이력서 분석 | 사용자의 이력서 텍스트 요약 및 핵심 역량 추출 |
| 채용 공고 추천 | 이력서와 공고 간의 적합성 판단 |
| 추천 사유 생성 | 단순 점수가 아닌 **자연어 설명 생성** |

- **활용 목적**
  - 정형화된 추천이 아닌, 설명 가능한 커리어 조언 제공

- **활용 방식**
  - Backend에서 프롬프트 구성
  - AI 응답을 API 형태로 클라이언트에 전달

- **장점**
  - 추천 결과에 대한 이해도 향상
  - 사용자 경험(UX) 개선

## 8. 기타 포함 사항

### 📌 프로젝트 특징
- AI 결과를 **DB에 영속화**하여 재사용 가능
- RSS의 '최대 20개 반환'이라는 정보 크기 제한을 보완하고자, git 또는 실행장소 이동에 따른 DB 데이터의 삭제 전 dumpdata & loaddata로 데이터 유지 및 AI의 추천 범위 확대

### 🚀 향후 개선 사항
- 사용자 피드백을 반영한 추천 정확도 개선
- 대규모 데이터 대응을 위한 DB 전환
- 추천 결과에 대한 사용자 평가 기능
